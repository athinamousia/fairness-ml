# Explainability

Explainability methods help interpret model decisions and understand the impact of features. This section covers:
- **Feature Importance**: Identifies which features most influence model predictions.
- **Visualization Tools**: Provides plots and charts to explain model behavior and fairness metrics.

---

For more details, see the respective Python modules in the `src/` directory:
- `explainability.py`: Provides explainability utilities.

Refer to the output plots and CSVs in the `output/` directory for results and visualizations.
